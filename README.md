##  Overview

**Cloud Hotel** is a user-centered flight booking application designed with **accessibility** and **minimalism** in mind. The project addresses common pain points in existing travel apps: crowded interfaces, confusing steps, and lack of transparency.

---

##  Problem Statement

Many flight booking apps are difficult to use because:
-  Screens are too crowded
-  Too many steps in the booking process
-  Unclear instructions and policies
-  Not accessible for users with disabilities
-  Hidden fees and confusing cancellation policies

---

##  Solution

A simple, transparent, and accessible flight booking app that:
-  Clear, minimalist interface
-  Streamlined booking process
-  Transparent pricing and policies
-  Accessibility features for all users
-  AI-powered assistance

---

##  User Research

| Metric | Value |
|--------|-------|
| **Participants** | 55 |
| **Method** | Google Forms Survey |
| **Key Insight** | 76.4% experienced problems with travel apps |
| **Top Priority** | Price (47/55 participants) |

### Key User Needs Identified:

| Need | Insight | Feature |
|------|---------|---------|
| Save favorite cities | Users frequently travel to same destinations | Favorite City feature |
| Real-time updates | Travelers want flight status info | Flight Tracking |
| Visa information | Users book without knowing visa requirements | Visa Requirement Checker |
| Price comparison | Users wait for better prices | Price Drop Alerts |
| Help during booking | Users have questions during process | AI Assistant |

##  Key Features

| Feature | Description |
|---------|-------------|
| **Favorite City** | Save and quickly access frequently traveled destinations |
| **Flight Tracking** | Real-time updates on delays, gates, boarding times |
| **Visa Checker** | Check visa requirements based on nationality & destination |
| **Price Alerts** | Get notified when prices drop for specific routes |
| **AI Assistant** | 24/7 chatbot for booking help and policy questions |
| **Class Comparison** | Clear comparison of economy, business, first class |
